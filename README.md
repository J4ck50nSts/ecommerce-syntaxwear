# 👟 SyntaxWear - E-commerce de Sneakers

A **SyntaxWear** é uma landing page de um e-commerce especializado em tênis e sneakers de alto padrão. O projeto foi desenvolvido com foco em um design moderno, limpo e responsivo, proporcionando uma excelente experiência de navegação para o usuário.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end para iniciantes:

- **HTML5:** Estruturação semântica de todo o conteúdo.
- **CSS3:** Estilização avançada com o uso de:
    - **Variáveis CSS:** Para facilitar a manutenção de cores e fontes.
    - **Flexbox & CSS Grid:** Para criar layouts flexíveis e alinhamentos precisos.
    - **Metodologia de Componentes:** Organização do CSS em arquivos separados (Header, Hero, Footer, etc) para melhor escalabilidade.
    - **Responsividade:** Design adaptável para dispositivos móveis e desktops.

---

## ✨ Funcionalidades

- [x] **Cabeçalho Funcional:** Navegação principal categorizada e ícones de acesso rápido (Carrinho, Usuário, Ajuda).
- [x] **Menu Mobile:** Implementado com a técnica de "Checkbox Hack" para garantir funcionalidade sem necessidade inicial de JavaScript.
- [x] **Seção Hero Impactante:** Banner principal com chamadas para ação (CTAs).
- [x] **Navegação por Categorias:** Cards interativos para categorias como Casual, Esporte, Moderno e Futurista.
- [x] **Grade de Produtos:** Vitrine de produtos organizada em grid, destacando o modelo principal "Krypton One".
- [x] **Rodapé Completo:** Com formulário de Newsletter, links de redes sociais e mapa do site.

---

## 📂 Estrutura de Pastas

```text
├── css/
│   ├── components/      # Estilos específicos de cada seção (Header, Footer, etc)
│   ├── base.css         # Estilos globais e tipografia
│   ├── layout.css       # Estrutura principal do site
│   ├── reset.css        # Reset de estilos padrões do navegador
│   └── variables.css    # Definição de cores e tamanhos (Design Tokens)
├── images/              # Ativos visuais, logos e ícones
├── index.html           # Arquivo principal do projeto
└── README.md            # Documentação do projeto
```

## 🔧 Como Rodar o Projeto

Como este é um projeto estático (HTML e CSS), é muito simples visualizá-lo:

1. Faça o download ou clone este repositório.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox, etc).
4. *Dica:* Se você usa o VS Code, recomendo usar a extensão **Live Server** para ver as alterações em tempo real.
